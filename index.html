✅ Zaliczenie – działa!

Ta strona jest serwowana z kontenera Nginx uruchomionego przez docker compose.

Uruchomione usługi

  • PostgreSQL (baza danych)
  • Adminer (panel WWW do bazy)
  • Portainer (panel WWW do Docker)
  • Nginx (ta strona)

Linki lokalne

  • Nginx: http://localhost:8080
  • Adminer: http://localhost:8081
  • Portainer: http://localhost:9000

Jeśli to widzisz – kontener web działa poprawnie.
